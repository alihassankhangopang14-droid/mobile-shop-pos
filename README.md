# 📱 Ali Mobiles & Communication - POS System

موبائل شاپ کے لیے مکمل POS سسٹم — انوینٹری، سیلز، ریپیرنگ کھاتہ، ایزی پیسہ/جاز کیش کیش بک، اور پرنٹ ایبل رسید۔

## 🚀 GitHub پر ڈالنے اور چلانے کا طریقہ

### مرحلہ 1: GitHub Repository بنائیں
1. [github.com](https://github.com) پر جا کر نیا **New Repository** بنائیں (مثلاً `ali-mobiles-pos`)
2. اسے **Public** رکھیں (Streamlit Cloud کے مفت ورژن کے لیے ضروری ہے)
3. یہاں سے تینوں فائلیں اپلوڈ کریں: `app.py`, `requirements.txt`, `README.md`
   - GitHub ویب سائٹ پر "Add file" → "Upload files" پر کلک کریں اور فائلیں ڈریگ کریں

### مرحلہ 2: Streamlit Cloud پر ایپ چلائیں (مفت)
1. [share.streamlit.io](https://share.streamlit.io) پر جائیں اور GitHub اکاؤنٹ سے لاگ ان کریں
2. **"New app"** پر کلک کریں
3. اپنی repository منتخب کریں، Main file path میں `app.py` لکھیں
4. **Deploy** پر کلک کریں — 2-3 منٹ میں آپ کی ایپ لائیو ہو جائے گی، ایک لنک ملے گا جو آپ کسی کو بھی بھیج سکتے ہیں

### مقامی کمپیوٹر پر ٹیسٹ کرنے کے لیے (اختیاری)
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📋 فیچرز
- **انوینٹری/خریداری:** موبائل خریداری، IMEI ٹریکنگ، فروخت کا اندراج
- **ریپیرنگ کھاتہ:** کسٹمر، خرابی، ایڈوانس/بقایا، اسٹیٹس (Pending/Ready/Delivered)
- **ایزی پیسہ/جاز کیش کھاتہ:** کیش ان/آؤٹ اور دکان کے اخراجات
- **پرنٹ ایبل رسید:** سیلز اور ریپیرنگ دونوں کے لیے پرنٹ ایبل بل
- **ڈیش بورڈ:** کل فروخت، منافع، اسٹاک مالیت، فعال ریپیرز

تمام ڈیٹا `ali_mobiles_pos.db` (SQLite فائل) میں خودکار طور پر محفوظ ہوتا ہے۔
